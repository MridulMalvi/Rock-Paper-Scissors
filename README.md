# Rock-Paper-Scissors-Game

A fun, interactive, web-based Rock Paper Scissors game built with HTML, CSS, and Vanilla JavaScript. Play against the computer bot, track your scores, and see who reaches 10 points first to become the champion!

## 🎮 Features

* **First to 10 Wins**: The game keeps track of the score and automatically declares a champion once either the user or the bot reaches a score of 10.
* **Dark / Light Mode**: Seamlessly toggle between Light and Dark themes for a better viewing experience using the mode button in the header.
* **Dynamic Feedback**: The game updates the message board in real-time, letting you know if you won, lost, or tied the round, along with the specific moves played.
* **Replay Functionality**: Once a game concludes, a replay button appears, allowing you to easily reset the scores and start a new match without refreshing the page.
* **Interactive UI**: Clean layout featuring hover effects, smooth transitions, and distinct color-coded victory/loss messages.

## 🛠️ Technologies Used

* **HTML5**: For structuring the web page and game layout.
* **CSS3**: For styling, flexbox layouts, hover effects, and responsive design.
* **JavaScript (ES6)**: For game logic, DOM manipulation, score tracking, and event handling.

## 🚀 How to Play

1. **Open the Game**: Simply open `index.html` in your preferred web browser.
2. **Make Your Move**: Click on one of the three images (Rock, Paper, or Scissors) to select your choice for the round.
3. **Bot's Turn**: The computer bot will automatically and randomly select its move.
4. **Check the Result**: The message box below the scoreboard will declare if you won, lost, or tied the round. 
5. **Win the Match**: Keep playing until either you or the bot reaches **10 points**.
6. **Play Again**: Once the game ends, the choices will lock. Click the **"Replay Game 🔁"** button to reset the board and start over!

## 📂 File Structure

* `index.html`: The main HTML document.
* `style.css`: The stylesheet containing all the visual styling and layout rules.
* `app.js`: The JavaScript file containing the core game logic and theme-toggling code.
* `rock.jpg`, `paper.jpg`, `scissor.jpg`: Image assets used for the player choices.
